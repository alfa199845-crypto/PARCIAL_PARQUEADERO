## Proyecto: Sistema de Parqueadero en C++

Este proyecto consiste en el desarrollo de un sistema de gestión de parqueadero que permite controlar la entrada y salida de vehículos, calcular el tiempo de permanencia y el valor a pagar.
---
## Innovaciones implementadas

- Numeración de parqueaderos
- Selección manual de espacios
- Validación de placas duplicadas
- Búsqueda de vehículos por placa
- Resaltado visual en el mapa (color verde)
- Historial completo de vehículos (entrada, salida, tiempo y pago)
- Mejora visual del mapa con alineación
